## About WISE

**WISE (Women in Software Engineering)** is an exclusive programme run in collaboration with **TalentSprint**, operating in parallel with the regular academic curriculum. Its primary aim is to prepare women engineering students to be industry-ready by making them proficient in the latest technologies.

## Program Objectives

- Enable students to develop mid-size software systems through experiential learning
- Develop an impressive resume for large multinational software organisations
- Build industry-necessary skills to make students confident and competent professionals

## Program Modules

WISE spans across the academic cycle with six lecture modules (36 hours each, one session per week per semester) and two project modules (60 hours each, completed during academic breaks).

### Lecture Modules

| Module | Content |
|--------|---------|
| Python 101 | Basic Python — precursor to programming (15 hrs) |
| Python 102 | OOPs, file handling, data structures, debugging |
| ML 201 | Machine Learning fundamentals, NumPy, SciPy, Anaconda |
| ML 202 | Advanced ML and AI techniques — perception, reasoning, learning |
| MEAN Stack 301 | AngularJS — full stack web applications, JWT authentication |
| MERN Stack 302 | React, Node.js, MongoDB — social network-style application |

### Project Modules (during academic breaks)

- **Python Project 103** — Modern Programming Practice using Python (summer break, 1st–2nd year)
- **ML Projects 203** — Machine learning projects with Flask (summer break, 2nd–3rd year)
- **AngularJS Projects 303** — Full stack application website projects (summer break, 3rd year)

## Branches Covered

CSE · IT · ECE · EEE · CSE AI&ML
