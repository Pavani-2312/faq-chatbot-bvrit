## Overview

The CSE Department has **6 well-equipped state-of-the-art laboratories** with the latest configuration systems. All systems are connected via LAN to the Domain Server (BVRITH). Labs are kept open even after college hours for student practice.

## Laboratories List

1. Operating Systems Lab
2. Database Management Systems (DBMS) Lab
3. Node JS / React JS / Django Lab
4. Data Structures Lab
5. Object Oriented Programming through Java Lab
6. Data Visualization — R Programming / Power BI
7. Machine Learning Lab
8. Artificial Intelligence Laboratory

## Lab Descriptions

### Operating Systems Lab
- Simulation of CPU scheduling algorithms, page replacement algorithms, memory allocation, and file allocation strategies
- Implementation of Unix system calls for process management and IPC

### DBMS Lab
- Uses MySQL database
- Practical exposure on designing, developing, and querying databases
- Covers ER data model, database design, normalization, SQL, procedures, cursors, and triggers

### Node JS / React JS / Django Lab
- Static web pages with HTML and client-side validation using JavaScript
- Java full stack development; Node JS for server-side programming
- Single Page Application development using React

### Data Structures Lab
- C programming, searching and sorting algorithms, stacks, queues, linked lists

### OOP through Java Lab
- Core Java using SDK 1.8 and Eclipse
- Abstract classes, Java Collection Framework, multithreading, GUI using Swing

### Data Visualization Lab (R Programming / Power BI)
- Business Intelligence using Tableau
- Dashboard applications, data pattern recognition, multiple visualization formats

### Machine Learning Lab
- Python-based ML techniques
- Predictive data analysis, model selection, evaluation of ML algorithms

### Artificial Intelligence Lab
- Basic principles of AI: problem solving, knowledge representation, learning

### Full Stack Development Lab
- Node JS, React, Express, Angular
- MongoDB for database operations
- Real-time application development

### IoT Lab
- Raspberry Pi platform
- LED, distance sensor, M2M protocols, Python scripting for IoT devices

### Scripting Languages Lab
- Ruby, Perl, TCL, and TK scripting languages
- Web-based project development

### Mobile Application Development Lab
- Android application development
- User interface design, URL-related applications

### Software Testing Lab
- Test strategies, GUI and database checkpoints
- Batch testing and automation

### Computer Networks Lab
- Data link layer, error detection/correction, routing protocols
- Network simulation tools

### DevOps Lab
- CI/CD pipelines, project management and integration tools
- Selenium for automated testing

### Flutter (UI Design) Lab
- Flutter widgets, layouts, responsive UI, navigation
- Themes, animations, form validation

### Cryptography and Network Security Lab
- DES, Blowfish, RSA, SHA-1, MD5 algorithms
- Diffie-Hellman Key Exchange

### Compiler Design Lab
- Lexical analysis, parsing (LL and LR parsers)
- Code optimization, intermediate code generation
