## Overview

The CSE AI&ML Department has **2 well-equipped state-of-the-art laboratories** with the latest configuration systems connected via LAN to the Domain Server (BVRITH). Labs remain open after college hours for student practice.

## Laboratories List

1. Programming for Problem Solving Lab
2. Data Structures Lab
3. Python Programming Lab
4. Database Management Systems (DBMS) Lab
5. Operating Systems Lab
6. Java Programming Lab
7. Machine Learning Lab
8. Computer Networks Lab
9. Artificial Intelligence and Natural Language Processing (AI & NLP) Lab
10. DevOps Lab
11. Deep Learning Lab
12. Node JS / React JS / Django Lab
13. Software Engineering Lab

## Key Lab Descriptions

### Machine Learning Lab
- Experimenting with ML models trained on datasets
- Classification and regression problems
- Python-based implementation of ML algorithms

### Artificial Intelligence and NLP Lab
- AI principles for problem solving using PROLOG
- Training and evaluating NLP models
- Design of innovative NLP applications

### Deep Learning Lab
- Foundation of deep learning and neural networks
- Deep Learning algorithms for various learning tasks
- Solving real-world problems with Deep Learning

### DevOps Lab
- CI/CD pipelines, Jenkins for continuous integration
- Project management, integration, testing and code deployment tools

### Computer Networks Lab
- TCP/IP layer protocol analysis using Wireshark
- Routing, error control, error detection algorithms
